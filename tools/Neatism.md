### File type supported
* Images
* Documents
* Archives
* Applications
* Torrents
* Fonts
* Videos
* Music
* DB Files
* Game Files
* Web Files

### Usage
* Edit Neatism.sh and add folders you want to sort in - 
  * paths_array=("path_1" "path_2")

* Terminal
  * run Neatism.sh.
* Cron Job
  * ex - 18 23 * * * cd path_to_script && ./Neatism.sh >/dev/null 2>&1
